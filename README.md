# My First Repo

## Мета
Навчитися працювати з Git та GitHub: створення репозиторію, коміти, гілки, злиття.

# Функціонал
- Репозиторій з базовою структурою
- Демонстрація роботи з гілками
- Використання SSH для доступу

## Структура
- `/src` — основні файли або код  
- `/docs` — документація  
- `README.md` — опис проєкту  
- `.gitignore` — виключення для Git
